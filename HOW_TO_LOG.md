## âœ… Validation & Error Prevention

### Automatic Checks:
1. **Total Calculation**:
   - App automatically adds: 50 + 40 + 10 = 100
   - No manual math needed!

2. **Max Set Detection**:
   - App finds the largest set: max(50, 40, 10) = 50
   - Automatically recorded

3. **Empty Sets Ignored**:
   - If you add a set but leave it blank, it's skipped
   - Only sets with numbers are saved

4. **Minimum Validation**:
   - Can't enter negative numbers
   - 0 is allowed (deletes entry if editing)

## ğŸ†˜ Troubleshooting

### "I entered sets but total is wrong!"
- **Check**: Is every set filled in correctly?
- **Solution**: Look at the green total card, verify math
- **Fix**: Edit any incorrect set, total updates automatically

### "I want to add a 4th set but can't find the button"
- **Check**: Did you scroll down in the modal?
- **Solution**: "+ Add Set" button is below your existing sets
- **Tip**: Modal is scrollable!

### "My max set isn't updating in Stats"
- **Check**: Did you use "Track Individual Sets"?
- **Reason**: Max set only tracked when breakdown is provided
- **Solution**: Edit entry, enable advanced mode, add sets

### "Can I enter sets without individual breakdown?"
- **Yes**: Use simple mode (just total)
- **Tradeoff**: Max single set won't be tracked
- **Note**: You can always edit later to add breakdown

### "I accidentally deleted an entry!"
- **Solution**: Tap that date again
- **Re-enter**: All the data (History doesn't auto-save on delete)
- **Tip**: Be careful with the trash button!
