# PWA Icon Instructions

To complete the PWA setup, you need to create two icon files:

1. **icon-192x192.png** - 192x192 pixels
2. **icon-512x512.png** - 512x512 pixels

## Design Guidelines
- Use a simple, recognizable icon (e.g., a bicep emoji, pushup silhouette, or "2026" text)
- Dark background (#000000) to match the app theme
- Bright green accent color (#4ADE80) for the main icon element
- Ensure good contrast for visibility on home screens
